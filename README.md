# Bellabeat Wellness Analysis

##  Project Description
This project is a data analysis case study completed using the **R programming language**.
The goal was to analyze Fitbit smart device data to identify trends in user activity
and sleep behavior, and to apply these insights to support data-driven marketing
recommendations for **Bellabeat**, a wellness technology company.

##  Business Task
- Analyze smart device usage data to uncover behavioral trends
- Evaluate how these trends relate to wellness habits
- Provide actionable marketing recommendations for Bellabeat products

##  Key Insights
- Users demonstrate consistent daily activity patterns with variation across weekdays
- Sedentary time significantly outweighs active time for most users
- Moderate to high daily activity is associated with improved sleep efficiency

##  Tools & Technologies
- **R programming language**
- RStudio (Posit Cloud)
- tidyverse, lubridate, ggplot2
- R Markdown for reproducible reporting

##  Repository Contents
- `BellabeatWellnessAnalysis.Rmd` – Reproducible R Markdown analysis
- `BellabeatWellnessAnalysis.html` – Final report with visualizations
- `figures/` – Generated plots 

##  Data Source
FitBit Fitness Tracker Data (CC0 Public Domain), provided via Kaggle (Mobius).

##  Outcome
The analysis supports Bellabeat marketing strategies focused on encouraging
sustainable daily activity, improving sleep quality awareness, and positioning
Bellabeat as a holistic wellness companion.
